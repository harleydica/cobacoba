## Point Konfigurasi

Pada soal UKK SIJA Paket 1 Mikrotik 2023 kita akan berjumpa dengan konfigurasi: 

1. VLAN (layer 3 & 2)
2. Wireless
3. DHCP Client
4. IP Address
   - VLAN 100 dengan network 192.168.100.0/24 sebagai VLAN admin.
   - VLAN 200 dengan network 192.168.200.0/24 sebagai VLAN jaringan.
   - Wlan1 Hotspot dengan network 172.16.17.0/24.
5. DHCP Server
6. Hotspot
   - Username: tamu
   - Password: guest
7. Firewall NAT

## Konfigurasi Router

Pertama dalam mengerjakan soal UKK ini adalah melakukan konfigurasi pada Router terlebih dahulu.

### Konfigurasi VLAN

Untuk membuat konfigurasi VLAN, pertama-tama kita harus membuat interface VLAN pada Mikrotik. Langkah-langkahnya adalah sebagai berikut:

1. Buka winbox dan masuk ke Mikrotik RouterOS.
2. Klik menu Interfaces dan pilih VLAN.
3. Klik tombol + untuk membuat interface VLAN baru.
4. Isikan ID VLAN sesuai dengan subnet yang diinginkan (contoh: VLAN 10 dengan ID 100).
5. Pada menu Interface, pilih interface yang akan digunakan sebagai trunk port (port yang menghubungkan router dengan switch).
6. Klik tombol OK untuk menyimpan konfigurasi.
