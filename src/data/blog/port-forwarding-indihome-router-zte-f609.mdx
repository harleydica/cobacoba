Apa sih port forwarding?? Pasti bagi kalian administrator jaringan tidak asing lagi dengan kata port forwarding, **port forwarding** merupakan salah satu konfigurasi router yang akan membuka akses perangkat lokal agar bisa diakses dari internet. Port forwarding itu sendiri hampir sama dengan DMZ pada modem, bedanya port forwarding hanya membuka akses sesuai dengan port yang kita buka dan bisa digunakan untuk banyak IP, DMZ tidak bisa digunakan dalam banyak IP tetapi akan membuka semua port yang tersedia. Kebanyakan administrator jaringan menggunakan port forwarding untuk mengaktifkan Webserver pribadi, CCTV Online dan mungkin untuk kebutuhan yang diakses melalui internet. Tidak hanya itu port forwarding bisa digunakan untuk **Mail Server, VPN, Remote Akses**.

Blog kali ini akan menjelaskan cara setting port forwarding pada perangkat ZTE F609 yang biasa disewakan oleh ISP Indihome, bagaimana caranya?? simak sampai selesai guyssðŸ˜‰
![Topologi](https://ik.imagekit.io/taufik/blog/port-forwarding/topologi.png 'Topologi')

Pastikan modem router yang kalian gunakan sudah mendapatkan IP Public dari ISP atau Layanan Internet, jika berlangganan Indihome kalian bisa restart modemnya hingga mendapatkan IP Public dari Indihome. Jika sudah kalian bisa melanjutkan tutorial ini sampai selesaiðŸ˜‰.

Pertama, buka router/ONT ZTE F609 Indihome biasanya dengan default IP [192.168.1.1](http://192.168.1.1), masuk dengan username admin dan password telecomadmin (default password ZTE F609) jika username atau password sudah diganti masuk dengan user yang telah diganti.
![Login Page](https://ik.imagekit.io/taufik/blog/port-forwarding/login-page.png 'Login Page')
![Tampilan Awal](https://ik.imagekit.io/taufik/blog/port-forwarding/page.png 'Tampilan Awal')

Setelah berhasil login, buka menu **Application => Port Forwarding**.
![Menu Port Forwarding](https://ik.imagekit.io/taufik/blog/port-forwarding/menu-port-forwarding.png 'Menu Port Forwarding')

Sebagai contoh ketika ingin memforward port 80 (HTTP) ke IP 192.168.1.201 Webserver yang berada di jaringan lokal.
![Port Forwarding](https://ik.imagekit.io/taufik/blog/port-forwarding/port-forwarding.png 'Port Forwarding')

Berikut yang perlu disini

- Enable => Checklist untuk menghidupkan Port Forwarding
- Name => Isikan nama apa saja bebas
- Protocol => Protocol jaringan yang digunakan, karena ingin forward HTTP maka gunakan TCP
- WAN Start Port => Port awal yang dibuka
- WAN End Port => Port akhir yang dibuka
- LAN Host IP Address => IP Local yang ingin diforward
- LAN Host Start Port => Port awal LAN yang dibuka
- LAN Host End Port => Port akhir LAN yang dibuka

Jika sudah klik **Add** untuk menyimpan port forwarding

Wah kita sudah melakukan port forwarding pada perangkat router ZTE F609 (modem Indihome), setiap request port 80 (HTTP) dari internet ke IP Public router maka akan di forward ke IP Lokal 192.168.1.201 yang menggunakan protokol HTTP:80. TERIMAKASIHðŸ˜‰
