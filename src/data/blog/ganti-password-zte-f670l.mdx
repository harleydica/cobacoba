## Pendahuluan

Apa itu ONT ZTE F670L?

ONT (Optical Network Terminal) ZTE F670L merupakan perangkat yang diberikan oleh MyRepublic untuk menghubungkan jaringan internet berbasis serat optik ke perangkat-perangkat di rumah Anda. ONT ini memungkinkan Anda untuk mengakses internet dengan kecepatan tinggi melalui koneksi serat optik.

![thumbnail](https://ik.imagekit.io/taufik/blog/ganti-password-zte-f670l/thumbnail.jpeg 'thumbnail')

Mengapa Anda Perlu Mengganti Password WiFi?

Mengganti password WiFi pada ONT ZTE F670L sangat penting untuk menjaga keamanan jaringan Anda. Password WiFi yang kuat dapat melindungi jaringan Anda dari akses yang tidak sah dan mencegah orang-orang yang tidak diinginkan untuk menggunakan koneksi internet Anda.

Persiapan Sebelum Mengganti Password WiFi:

Sebelum memulai proses mengganti password WiFi pada ONT ZTE F670L, pastikan Anda memiliki informasi berikut:

1. Kredensial Masuk: Pastikan Anda memiliki akses ke halaman administrasi ONT. Biasanya, Anda memerlukan alamat IP tertentu dan kredensial masuk (username dan password) yang diberikan oleh penyedia layanan internet (MyRepublic).
    - Terdapat dua user kredensial ONT ini. 
    - User biasa 
      - username **user**
      - password **user**
    - Super USER
      - username **admin**
      - password **Web@0063**
    - Silahkan coba kedua kredensial diatas, secara default ONT F670L myRepublic memiliki kredensial seperti diatas, jika salah bisa hubungi Customers Service myRepublic.
2. Password Baru: Siapkan password baru yang kuat untuk digunakan sebagai password WiFi baru Anda.

## Langkah-Langkah

Pertama buka browser chrome / firefox, lalu ketikkan alamat IP dari modem ZTE F670L biasanya **192.168.1.1**

![pertama](https://ik.imagekit.io/taufik/blog/ganti-password-zte-f670l/pertama.png 'pertama')

login menggunakan kredensial: 
- User biasa 
  - username **user**
  - password **user**
- Super USER
  - username **admin**
  - password **Web@0063**
- Silahkan coba kedua kredensial diatas, secara default ONT F670L myRepublic memiliki kredensial seperti diatas, jika salah bisa hubungi Customers Service myRepublic.

Lalu klik <Kbd>Login</Kbd>.

Setelah berhasil login, silahkan masuk ke menu, <Kbd>Local Network</Kbd> lalu <Kbd>WLAN</Kbd> seperti pada gambar di bawah ini. 

![kedua](https://ik.imagekit.io/taufik/blog/ganti-password-zte-f670l/kedua.png 'kedua')

Untuk merubah SSID atau Nama Wifi dan passwordnya, Anda bisa masuk ke menu <Kbd>WLAN SSID Configuration</Kbd>.

![ketiga](https://ik.imagekit.io/taufik/blog/ganti-password-zte-f670l/ketiga.png 'ketiga')

Selanjutnya Kalian tinggal ubah SSID1 yaitu WiFi dengan frekuensi 2,4Ghz, lalu pada **SSID Name** yaitu nama WiFi ubah sesuai dengan keinginanmu, lalu **WPA Passphrase** itu adalah Password WiFi nya, silahkan isikan password sesuai keinginanmu.

Kalian juga perlu mengganti nama WiFi dan Password WiFi 5Ghz pada SSID5 dengan cara yang sama seperti diatas.

![Keempat](https://ik.imagekit.io/taufik/blog/ganti-password-zte-f670l/keempat.jpg 'keempat')


<Callout type='warning'>
Perlu diingat untuk nama WiFi SSID1 dan SSID5 harus sama saat mengganti password WiFi, karena jika berbeda akan berakibat tidak bisa terhubung pada WiFi Kalian.
</Callout>

Selamat kalian sudah berhasil mengganti nama atau password WiFi.

<Callout type='warning'>
Perlu diingat untuk password WiFi yang ingin dirubah minimal memiliki 8 digit (angka,huruf,simbol). Password tidak boleh lupa atau salah ketik, karena jika terjadi kesalahan dalam penulisan Modem WiFi perlu di restart pabrik untuk memulihkan password WiFi seperti semua. Atau bisa menghubungi Customers Service myRepublic / ISP untuk mengganti password WiFi Anda.
</Callout>
